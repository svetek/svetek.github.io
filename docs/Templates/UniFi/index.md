Use this section for reusable templates related to UniFi deployments and operations. UniFi has multiple product families, so templates are organized by product area instead of being stored in one flat list.

## Product Areas

| Product Area | Templates |
| --- | --- |
| [UniFi Access](Access/access-control-rollout-templates/) | Building access rollout emails, user handouts, emergency procedures, and admin cheat sheets |
| UniFi Protect | Camera, NVR, retention, and access-review templates can be added here |
| UniFi Network | Network, Wi-Fi, VLAN, and site-change templates can be added here |
| UniFi Endpoint | Endpoint rollout and operations templates can be added here |
| UniFi Console | Console ownership, backup, admin access, and lifecycle templates can be added here |

## Template Rules

- Keep public templates generic.
- Replace all bracketed placeholders before sending to a client.
- Store completed client copies in the client's documentation workspace.
- Do not publish access codes, reader names, unit rosters, admin accounts, or emergency contact details.
